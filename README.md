# tech @ vernacular.ai

This is source for the [tech team webpage][tech_blog] at Vernacular.ai. Template is [Mundana by WowThemes.net][mundana]

## Developing

- Clone the repository
- `bundle`
- Run the blog locally: `bundle exec jekyll serve --host=0.0.0.0`
